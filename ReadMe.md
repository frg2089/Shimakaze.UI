## 支持的操作系统

| 操作系统  |  运行环境   |         WebView          |
| :-------: | :---------: | :----------------------: |
|  Windows  |    Win32    | Edge WebWiew2[^webview2] |
| GNU/Linux | Gtk4[^gtk4] | WebKitGtk6[^webkitgtk6]  |

[^webview2]: [下载 Edge WebView2](https://developer.microsoft.com/en-us/microsoft-edge/webview2/)
[^gtk4]: libgtk-4.so.1
[^webkitgtk6]: libwebkitgtk-6.0.so.4
